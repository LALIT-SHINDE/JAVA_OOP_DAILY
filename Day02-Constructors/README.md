DAY02-Constructors
Objective:
     - Understand how constructors work in Java and how they are used to initialize objects automatically.

Concepts Covered:
  - Default Constructor
  - Parameterized Constructor
  - Constructor Overloading

Program Explanation:
  - In this program, we created a Student class with:

Attributes:
name
age
Constructors:
  - Default Constructor → assigns default values (Unknown, 0)
Parameterized Constructor → assigns user-defined values
Method:
display() → prints student details

Sample Output:
Name: Unknown  
Age: 0  
Name: Siddharth  
Age: 54  

Key Learnings:
  - A constructor is called automatically when an object is created.
It helps initialize object data efficiently.
Multiple constructors can exist in a class (constructor overloading).
Constructor name must be the same as the class name.
Constructors do not have a return type
