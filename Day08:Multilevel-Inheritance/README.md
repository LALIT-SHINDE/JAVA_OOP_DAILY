Employee Management Hierarchy:

A Java implementation demonstrating Object-Oriented Programming (OOP) principles, specifically **Inheritance** and **Method Overriding**.

Overview:
This project defines a multi-level hierarchy for organizational roles, ensuring efficient data handling and modularity.

Class Structure:
- Employee: The base class containing core identity markers (Name, DOB, ID).
- Manager: Inherits from Employee and adds financial data (Salary).
- SalesManager: Inherits from Manager and adds performance-based incentives (Commission).

 How to Run
1. Ensure you have the Java Development Kit (JDK) installed.
2. Compile the classes:
   
   javac Main.java
